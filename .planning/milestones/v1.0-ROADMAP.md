## Milestone Summary

**Key Decisions:**
- Pure HTML/CSS (no frameworks) — fast load, easy deploy, SEO-friendly
- GitHub + Vercel deployment — free tier, instant deploys
- Formspree for form handling — works with static HTML, no backend needed
- GA4 only (no GTM) — lightweight for static site
- Custom 404.html — keeps visitors on-site

**Issues Resolved:**
- Form submission without backend (solved via Formspree)
- CSS minification achieved 41-44% size reduction
- All pages pass mobile-friendly viewport requirements

**Technical Debt:**
- GA4 measurement ID placeholder needs user-provided ID
- Formspree form ID placeholder needs user-provided ID
- Image optimization (actual Unsplash URL params) — currently placeholder w/h values
