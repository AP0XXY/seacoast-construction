| Property | Value |

---

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Border-radius (buttons), form label margin-bottom |
| sm | 8px | Compact element gaps, card image height adjustments |
| md | 16px | Standard element spacing, section internal gaps |
| lg | 24px | Section padding (horizontal), card padding, stats gap |
| xl | 32px | Stats section vertical padding, hero paragraph spacing |
| 2xl | 48px | Footer top padding, stat-number font size (40px approx) |
| 3xl | 64px | Hero section vertical padding, major section breaks |

Exceptions: none — all existing spacing in index.html already follows multiples of 4.

---

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body / Label | 16px (1rem) | regular (400) | 1.6 |
| Card heading | 20px (1.25rem) | bold (700) | 1.3 |
| Section heading | 32px (2rem) | bold (700) | 1.2 |
| Hero heading | 40–64px (2.5–4rem responsive) | bold (700) | 1.2 |

Heading hierarchy: Single H1 per page (hero), H2 for section titles, H3 for card/item headings, H4 for footer headings. Labels (form labels, nav links, footer text) use body size (16px) at regular weight.

---

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #ffffff (`--white`) | Page backgrounds, content sections, card surfaces |
| Secondary (30%) | #0b1426 (`--navy`) | Header, footer, dark overlays on hero images, text-dark for nav |
| Accent (10%) | #2f80ed (`--blue`) | CTA buttons, links, stat numbers, form focus border, hover states |
| Text muted | #666666 | Paragraph text on light backgrounds, secondary labels |
| Success | #28a745 | Form success message (green) |

Accent reserved for: CTA buttons (`.btn-primary`), hyperlinks (`.service-card a`), stat numbers (`.stat-number`), form focus border, nav hover states. Never for body text or backgrounds.

## Service Page Template Contract

All 6 service pages follow this identical section structure:

| Order | Section | CSS Class | Description |
|-------|---------|-----------|-------------|
| 1 | Header | `header`, `.header-inner` | Sticky nav with logo, nav links, phone, CTA — identical across all pages |
| 2 | Hero | `.hero` | Full-width background image with overlay, H1 with primary keyword, subtitle, two CTAs |
| 3 | Problem/Solution | `.problem-solution` (new) | Two-column or stacked: left = problem framing, right = solution approach |
| 4 | Process Steps | `.process` (new) | 4-step horizontal/vertical grid with step numbers, titles, descriptions |
| 5 | Benefits | `.benefits` (new) | 2–3 column grid of benefit items (icon/number + title + description) |
| 6 | Before/After Gallery | `.gallery` (new) | CSS grid of 3–4 images with consistent aspect ratio, `<!-- SWAP: real photo -->` comments |
| 7 | Mid-page CTA | `.mid-cta` (new) | Full-width navy background strip with CTA button centered |
| 8 | Related Services | `.related-services` (new) | 2–3 cards linking to related service pages |
| 9 | Contact Form | `#contact`, `.contact` | Reused from index.html — consultation form with service dropdown pre-selected |
| 10 | Footer | `footer` | Identical across all pages — same as index.html |

## Responsive Breakpoints

| Breakpoint | Layout Changes |
|------------|----------------|
| < 768px | Single column, hamburger nav, stacked form rows, stacked gallery grid |
| 768px–1023px | 2-column grids, hamburger nav, 2-column form rows |
| 1024px–1439px | Full desktop nav visible, 3-column grids where applicable |
| ≥ 1440px | Container max-width 1300px (consistent with homepage) |
